# Team

`scvi-tools` was started at the Yosef Lab at UC Berkeley with the goal of reducing the barrier to entry for using probabilistic models in single-cell omics analyses. We are grateful to have received many code contributions on GitHub [(scvi-tools graph)](https://github.com/YosefLab/scvi-tools/graphs/contributors).
